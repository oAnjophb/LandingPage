My Projects

AquaSense é uma plataforma inteligente para monitoramento, gestão e economia de água, com integração IoT, análise preditiva e dashboards interativos. Ideal para residências, condomínios, empresas e órgãos públicos.

Este projeto implementa o back-end e o front-end de um sistema para gerenciamento de fila de atendimento de pacientes em um pronto-socorro. O sistema permite o registro de pacientes, triagem, organização da fila e chamada para atendimento, com interface web moderna e responsiva.

G.R.A.F. — Gráficos Automáticos por Formato — é um aplicativo desktop desenvolvido em Python que transforma planilhas do Excel em gráficos automáticos e organizados. Ao selecionar um arquivo .xlsx, o sistema analisa seus dados, identifica categorias como meses ou anos, cria pastas específicas para cada uma e gera gráficos com base nas colunas e respostas presentes na planilha. Tudo é salvo de forma separada e intuitiva, facilitando a visualização e análise de dados com apenas alguns cliques.
